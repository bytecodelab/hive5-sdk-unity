# Hive5 SDK for Unity

아쉽게도, 아직 제대로 된 SDK 패키지가 아닙니다.
SDK를 포함한 Unity 샘플 프로젝트입니다.

Assets/Hive5 폴더만 복사하여 자신의 Unity 게임에 덮어쓰기 하시면
SDK를 이용하실 수 있습니다.
